# SHWMS
Smart Health and Wellness Monitoring System
> *For A Change.*

**Short Project Description**
This project helps users manage their daily health status.
It is a simple and easy-to-use application with a text-based interface suitable for users of all ages.
It also helps to maintain a healthy lifestyle by suggesting various exercises and activities as per the user's health status.
---

## Features

- Track daily health records (up to 7 days)
- Generate BMI-based diet plans
- Weekly gym exercise recommendations
- Daily health tips
- Demo mode for exploring the application

## Tech Stack

* **Programming Language:** Java
* **IDE:** IntelliJ IDEA (or any other IDE of your choice)
* **JDK:** 17.0.18

---

## How to Run

### Prerequisites

* Java JDK (Version: 17.0.18)
* IntelliJ IDEA (optional)

### Steps
1. Clone or download the repository.
2. Open the project in IntelliJ IDEA.
3. Build the project.
4. Run the main class(PROJECT_LOGIN_MENU).
